which css type have high priority inline, internal or external css?

Within these three ( Inline, Internal & External ) the priority order is first Inline, then Internal and last priority is given to external styles. This is the biggest advantage as we can override the global style property and define them locally. Let us start learning each of these three types of CSS.

why we use external css professionally

By applying consistent formatting to website pages, external style sheets help in bringing a uniform, global look and feel to a website. The external style sheet can be linked from HTML pages. When using an external style sheet, styles need to be set up only once for each element.
